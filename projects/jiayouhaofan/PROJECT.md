## 项目背景

评估对象：围绕"健康饮食 / 营养膳食 / 家庭营养健康管理"的短视频与私域转化项目。
暂定品牌名"家有好饭"，口号"爱家，从好好吃饭开始"。

定位：面向家庭场景的健康生活方式引导，核心是科学饮食、家庭膳食结构调整、行为陪跑。
边界：不替代医疗、不做诊断、不做治疗承诺、不夸大慢病逆转。

第一阶段目标客群：中老年人及其家庭（慢病风险、体检指标异常、控糖控脂控盐控体重需求、
子女替父母操心饮食的人群）。后续可拓展：孕产妇营养、儿童营养、体重管理、企业健康、
社区营养服务。

初步商业化设想：99 元家庭营养健康评估 → 399 元 21 天家庭饮食改善营 →
2999 元季度家庭营养顾问 → 会员 / 复购 / 转介绍 / 产品推荐。

3. 90 天运营计划、团队分工与数据指标

## 关键约束

- **预算上限**：30 万美元（约合 215 万元人民币），第一阶段全部投入
- **每周总工时**：200 小时（≈5 个全职当量）
- **团队定位**：中小团队，不依赖大厂级资源、明星资源、MCN 矩阵化人海战术
- **投流策略**：保守投流，第一阶段投流预算参考区间为总预算的 10%—20%
- **资质与合规预设**：需要的资质全部按合规办理，严禁挂靠、买证、擦边规避审核
- **地域与性质**：中国，全国线上业务，受客户委托制作
- **数据铁律**：所有数据须可核验来源、可推算或已登记为预设假设，禁止一切形式编造
- **独立性原则**：客户付费购买的是准确判断，不是乐观结论；
  "不建议立项"是合格交付物，迎合性结论是不合格交付物

完整执行细则见 `input/家有好饭_全自动执行提示词_v2_6_ClaudeCode版.md`。
